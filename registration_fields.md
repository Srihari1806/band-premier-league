# Kalakshetra Registration Fields & Details Collected

This document provides a comprehensive overview of all the details, files, and contact specifications collected from candidates when registering under different roles/categories for the **Kalakshetra** platform.

---

## 1. Bands & Solo Artists
This category gathers information about musical acts, their performance style, technical requirements, media assets, and key contacts.

| Field / Data Point | Type | Description |
| :--- | :--- | :--- |
| **Artist / Band Name** | String | Name of the musical act |
| **Tagline** | String | Short catchphrase describing the band/artist |
| **Genre** | Dropdown | Selection (Rock, Indie, Folk, Metal, Pop, Alternative, Hip-Hop, or Custom) |
| **Home City** | String | City where the act is based |
| **Formed In** | String | Year the band/artist started performing |
| **Languages** | String | Languages in which the act performs |
| **Content Type** | Selection | Focus of setlist (Originals, Covers, or Both) |
| **Biography** | Text Area | Broad description of the act, history, and style |
| **Musical Mission** | Text Area | The artistic goal or vision of the band |
| **Musical Influences** | Text Area | Primary bands, genres, or musicians that inspire them |
| **Sonic Identity** | Text Area | General description of their live sound and style |
| **Achievements** | Text Area | Notable career highlights, awards, or press mentions |
| **Profile Photo** | File Upload | Avatar/Logo image for listings (compressed under 2MB) |
| **Banner Image** | File Upload | Landscape background image for profiles (compressed under 2MB) |
| **Instagram Link** | URL | Official Instagram profile URL |
| **YouTube Channel** | URL | YouTube channel or video showcase URL |
| **Spotify Artist Page** | URL | Official Spotify link |
| **JioSaavn Page** | URL | JioSaavn profile link |
| **Apple Music Page** | URL | Apple Music artist profile link |
| **Website Link** | URL | Official band website |
| **Audio Demo Track** | File Upload | Direct demo recording (base64 audio file) |
| **Stage Rider** | File Upload | Technical stage layout sheet (tech rider) |
| **Technical Specs** | Text Area | Additional requirements (mics, power, amps, monitor mixes) |
| **Touring Cities** | String | Cities preferred/open for booking |
| **Performance Fee** | String | Standard pricing/compensation range per gig |
| **Contact Name** | String | Primary booking contact / spokesperson name |
| **Contact Phone** | String | Phone number for bookings and OTPs |
| **Contact Email** | String | Registered email address |
| **Manager Name** | String | Manager's name (optional) |

---

## 2. Venues & Cafes
This category collects specific venue sizes, logistics, electrical setups, and hospitality amenities to ensure bands are matched with suitable performance spaces.

| Field / Data Point | Type | Description |
| :--- | :--- | :--- |
| **Venue Name** | String | Name of the cafe, pub, bar, or auditorium |
| **Physical Address** | String | Complete address details |
| **Google Maps Link** | URL | Map pin location link |
| **Owner / Company** | String | Licensing or management entity name |
| **Venue Logo** | File Upload | Circular logo image (under 2MB) |
| **Cover Photo** | File Upload | Banner showcase of the space (under 2MB) |
| **Venue Type** | Dropdown | Category (Cafe, Pub, Restaurant, College, Open-air, or Indoor) |
| **Audience Capacity**| String | Maximum capacity range (e.g. 100-300) |
| **Stage Dimensions** | String | Approximate dimensions of the performance stage |
| **Facility Badges** | Checkboxes | Available features: Parking, Green Room, Sound System, Lights, Power Backups, In-house Food, Bar License |
| **Sound/Light Specs** | Text Area | Technical details of the in-house PA and lighting rig |
| **Gallery Photos** | Multi-File | Up to 4 interior/stage photos (under 2MB each) |
| **Available Days** | Checkboxes | Days open for scheduling gigs (Weekdays, Weekends, etc.) |
| **Preferred Genres** | String | Specific music styles aligned with the venue's vibe |
| **Revenue Terms** | String | Financial/rental model details (Ticket Share, Flat Rent, etc.) |
| **SPOC Name** | String | Single Point of Contact name |
| **SPOC Phone** | String | Phone number for scheduling and OTP |
| **SPOC Email** | String | Email address for booking confirmations |

## 3. Labels & Production Houses
This category handles agencies, record labels, and sound/visual production companies partnering to coordinate curation, promotion, or high-end sound execution.

| Field / Data Point | Type | Description |
| :--- | :--- | :--- |
| **Company Name** | String | Name of the label or production house |
| **Company Profile** | Text Area | Business overview, history, and production capabilities |
| **Represented Genres**| String | Primary music genres focus |
| **Support Options** | String | Investment size or promotional resources offered to acts |
| **Past Showcases** | Text Area | Artists or major releases previously supported |
| **Portfolio Link** | URL | Link to case studies, past releases, or video showreel |
| **Official Website** | URL | Official company website URL |
| **Instagram Link** | URL | Instagram profile handle link |
| **Operating Cities** | String | Areas of operations and distribution networks |
| **SPOC Contact Name** | String | Company point of contact name |
| **SPOC Phone** | String | Phone number for security login and OTP |
| **SPOC Email** | String | Company email for operational updates |
| **Company Logo** | File Upload | Square/circular company logo (under 2MB) |

| **Contact Email** | String | Corporate email address |

| Field / Data Point | Type | Description |

| **Past Campaigns** | Text Area | Examples of collaborative brand activations |

| **SPOC Email** | String | Email address |

## 6. Volunteers & Creative Crew

| Field / Data Point | Type | Description |
| :--- | :--- | :--- |
| **Full Name** | String | Name of the volunteer/creative |
| **Role Type** | Dropdown | Category (Ambassador/Volunteer, Music Producer, Videographer, Photographer, Podcast Partner, College Partner) |
| **Skills / Tech Stack**| Text Area | Specialized tools used (e.g. Premiere, FL Studio, Mixer consoles) |
| **City** | String | Base city of operation |
| **Availability** | String | Time allocation (e.g. Weekends, 10 hrs/week) |

| **Statement of Intent**| Text Area | Why they want to volunteer for Kalakshetra |
| **Profile Photo** | File Upload | Avatar/Face portrait (under 2MB) |
| **Contact Phone** | String | Mobile number |
| **Contact Email** | String | Email address |

## 7. Event Managers
Tracks independent gig organizers and agencies partnering to manage match-day crowds, ticketing entry, security, and stage management.

| Field / Data Point | Type | Description |
| :--- | :--- | :--- |
| **Agency / Name** | String | Event management agency or individual organizer name |
| **Operating Cities** | String | Cities where they have local permissions and vendor connections |
| **Past Event Gigs** | Text Area | Music festivals, concerts, or club gigs managed |
| **Core Team Size** | String | Size of the event logistics crew |
| **Manager SPOC** | String | Lead coordinator name |
| **SPOC Phone** | String | Phone number |
| **SPOC Email** | String | Email address |
